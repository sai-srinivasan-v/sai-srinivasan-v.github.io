Hybrid Cascade Router

work in progress

A provider-agnostic, multi-stage LLM routing package I'm actively building. It sends requests to a cheap fast tier first and escalates to stronger models only when a pluggable strategy — confidence-based, rule-based, or a hybrid of both — decides the cheap tier's answer isn't good enough. Currently has adapters for OpenAI, Anthropic, and local open-source models via Ollama, with a full test suite.
